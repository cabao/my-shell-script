# my-shell-script
收集一些個人常用的工具，寫成 script 更快部署  

OS : Debian/Ubuntu16.04  
## env_init.sh
|pkgs|description/source|
|-------------|-------|
|curl|傳輸工具|
|git|版本控制|
|aptitude|套件管理|
|Atom|文字編輯器|
|Java8|含JRE/JDK/Java Browser Plugin|
|Chrome|網頁瀏覽器|
|vim|文字編輯器|
|htop|系統監測|
|shutter|截圖|

## Setup
```
git clone https://github.com/cabao/my-shell-script.git
cd my-shell-script
./env_init.sh

```
